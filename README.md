# Generate your own project boilerplate with NodeJS

## Clone Repo

```
git clone https://github.com/rahatkabir007/boilerplates-cli-rkc
```

## Install Dependncies 

```
npm install 
```

## Install this package globally in your system

```
npm link 
```
